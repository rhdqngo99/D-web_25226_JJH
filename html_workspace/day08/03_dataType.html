변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a number : 숫자가 아님)
  • 문자 : String => "",'',`` 감싼 값들
  • 불리언 : Boolean => true / false
  • 객체 : Object => 1개 이상의 데이터가 특정 구조를 가지고 있는 형태 {키 : 값} {} : 아무것도 없는 사실도 1로 인정 1개의 객체로 취급
  • 배열 : Array => [1,2,3,4] 형태의 집합 (java에서의 list와 비슷한 형태 index, length 가 있음.)
  • 알수없는 값 : null => 인지가 안됨
  • 미지정 값 : undefined => 값이 할당되지 않음.
  1. 기본 자료형 : 값이 메모리의 주소값을 가르키는 곳에 직접 할당이 되어있는 값 number, boolean, string(참조자료형이지만 편의상 기본자료형처럼 사용)
  2. 참조자료형 : 값이 저장되어 있는 주소를 할당 string, array, object, function, null, undefined
  3. == 같다(값만 비교) 123 == '123' != 같지않다 === 같다 (값과 타입을 같이 비교) 123 === '123' !== 같지않다
