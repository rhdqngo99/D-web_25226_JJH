input, textarea, select, html element 텍스트를 받아오기

  • input tag는 value 속성에 값이 맵핑 되어 있어서 value 속성으로 값을 가져올 수 있음

  • 여는 태그 닫는 태그 사이에 값을 가져올 때 innerText
  • select => opion / radio / checkbox value의 값을 태그 내 속성으로 부여해주어야 선택한 값을 가져올 수 있음.
  • value / innerText 가져오는 값은 모두 string 처리됨.
  • string 처리된 값을 숫자로 사용하기 위해서는 변환이 필요함
