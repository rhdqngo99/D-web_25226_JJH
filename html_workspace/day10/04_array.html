자바스크립트에서 배열을 유동배열 : 크기, 데이터 변경이 가능

  • const myArray = [1,2,3,'a','b',{},[]];
  • 배열의 데이터 타입을 다르게 넣어도 가능. 잘 쓰지는 않음. => 처리가 복잡해지기 때문. 일반적으로 같은 타입의 데이터를 채우는것이 룰
  • const members = [{},{},{},{}];
  • myArray=[] : 빈 배열을 만든 후 데이터를 추가하여 사용
  • index를 가짐 => 0부터 시작
  • length => 배열의 총길이
  • const arr = new Array(1,2,3); // new 로 객체 생성이 가능하지만 잘 사용하지 않음
