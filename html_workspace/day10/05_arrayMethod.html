배열관련 메서드 : 유사배열에서는 사용제한이 되는 메서드 들이 있음. const myArray = [1,2,3,'a','b',{},[]]; // 유사배열(혼합배열)

  • array.push(value) : 배열의 마지막 위치에 value 추가 => 배열의 길이를 리턴
  • array.pop() : 배열의 마지막 요소를 제거 => 제거요소를 리턴
  • array.shift() : (pop()의 반대) 배열의 첫 요소를 제거 => 값을 리턴
  • array.unshift(value) : 배열의 첫 요소에 value를 추가 => 길이를 리턴
  • array.join(value) : 배열의 값에 value를 추가하여 하나의 문자열로 리턴
