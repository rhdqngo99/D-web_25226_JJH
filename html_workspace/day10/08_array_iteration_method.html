Array Iteration Method : 순환구조에서 사용하는 메서드

  • array.reduce(callback function(total, value, index, self),[init value]); 연산의 누적값을 리턴함.
  • array.every(callback function(value, index, self)) 콜백 함수의 리턴 조건에 모든 배열의 값이 만족하면 true / false 리턴 (AND)
  • array.some(callback function(value, index, self)) 콜백 함수의 리턴 조건에 하나의 값이 만족하면 true / false 리턴 (OR)
  • array.indexOf('str',[start index]); 해당 문자열이 존재하는 index를 리턴
  • array.lastIndexOf('str',[start index]) : 뒤에서 부터 검색
  • array.find(callback function(value, index, self)) 검색 조건에 만족하는 첫번째 값을 리턴
  • array.findIndex(callback function(value, index, self)) 검색 조건에 만족하는 첫번째 번지를 리턴
  • array.form('객체') : 순서가 있거나, 순환구조를 갖고 있는 객체를 배열로 변환
  • array.keys() : 배열에서 사용중인 key(index)를 검색하여 새로운 배열로 리턴
  • array.entries() : index:value 의 구조값을 만들어 새로운 배열에 담아 리턴
  • array.includes('str') : 배열에 str 값이 있으면 true / false
