비표준 속성(Non_standard attribute)

  • 속성 : attribute element에 달리는 여러가지 속성(빨간색)
  • 모든 element마다 본인의 속성을 가지고 있음. == 표준속성
  • element 속성이 제한적이고, 속성자체가 특정 기능을 가지고 있음. 개발자 입장에서 필요한 속성을 직접 제공하는 방법
  • 너무 과도하게 사용하면 개발 효율과 가독성 저하, 적당히 사용할 것.
  • 태그의 속성 이름을 data-변수명="값" 값이 리턴 됨 dataset property를 사용하여 읽어들임
  • 변수명에 대문자는 허용안함. 대문자로 쓰더라도 소문자로만 인식 (대문자 => 소문자로 변환해서 인식) let nameNum = dataset.nameNum; 인지 불가능 let nameNum = dataset.namenum; 인지 가능
