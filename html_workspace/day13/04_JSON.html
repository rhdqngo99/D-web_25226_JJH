JSON(JavaScript Object Notation) : 자바 스크립트 객체 표기법

  • 데이터 요청 및 응답처리에 필요한 데이터의 표준 형식을 제공하는 타입
  • 백엔드(서버)언어 종류에 상관없이 데이터를 송수신 단말형식으로 통일
  • 서버측에서는 무조건 String 처리 - 화면(javaScript)에서는 Object로 처리
  • 서버로 보낼때 > Object > string => JSON.stringify(jsObject);
  • 서버로 부터 화면에서 받을 때 > string > Object => JSON.parse(Stringdata);
