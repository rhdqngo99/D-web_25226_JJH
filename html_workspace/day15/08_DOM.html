DOM : Document Object Model : html 문서 객체 모델

  • html 하위에 있는 모든 태그들(element) / 속성(attribute) = 프로퍼티(property) 을 각지고 있는 문서 객체
  • element 가져올때 : getElementById , byClassName, byTagName
  • getElementById : 단일 엘리먼트 리턴
  • byClassName / byTagName : 복수 엘리먼트 리턴 => HTMLCollection 타입으로 리턴
  • HTMLCollection 타입 배열처럼 리턴 index를 사용하여 선택.
